# Libraries for AAP
Libraries are published to jitpack.

Example usage: 
```kotlin
repositories {
  maven("https://jitpack.io")
}
dependencies {
  implementation("com.github.navikt.aap-libs:kafka:0.0.7")
  implementation("com.github.navikt.aap-libs:kafka-test:0.0.7")
}
```
